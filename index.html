vk2tg

Утилита, которая пересылает посты из выбранной VK-группы в Telegram-канал.

О приложении

vk2tg — это небольшое Go-приложение, которое обращается к API VK, забирает последние публикации со стены группы и отправляет их в закрытый канал или чат Telegram. Утилита полезна, если вам нужно синхронизировать контент между платформами без ручного копирования.

Основные возможности

  • Получение свежих постов с помощью `wall.get`.
  • Формирование читабельного сообщения с ссылкой на оригинал.
  • Отправка через стандартный эндпоинт Telegram `sendMessage`.

Как использовать

Подготовьте токен VK с доступом к стене, ID нужной группы, токен бота Telegram и ID канала/чата. Запустите утилиту через go run ./cmd/vk2tg, передав параметры с помощью переменных окружения или флагов командной строки.
